## Installation

First, install [Yeoman](http://yeoman.io/) and `generator-creact-app` using [npm](https://www.npmjs.com/) (we assume you have pre-installed node.js).

```bash
$ npm install -g yo
$ npm install -g generator-creact-app
```

Then generate your new ReactJS App:

```bash
$ yo creact-app
```
